# QuizMaster - 智能错题本 v2.0

一个跨平台的智能错题管理系统，支持 Web 和移动端。

## ✨ 功能特性

### 🌐 Web 版 (`/web`)
- **OCR文字识别** - 上传图片自动识别错题内容（Tesseract.js）
- **AI智能分析** - 自动分析错因，给出学习建议
- **拍照记录** - 支持图片上传，快速记录错题
- **分类归档** - 按科目（数学/英语/语文/物理/化学）智能分类
- **智能组卷** - 根据错题自动生成复习试卷
- **PDF导出** - 一键导出试卷为PDF格式
- **复习提醒** - 基于艾宾浩斯遗忘曲线的复习提醒
- **学习统计** - 实时追踪学习进度和掌握情况
- **本地存储** - 浏览器 LocalStorage，保护隐私

### 📱 Flutter 版 (`/cuoti_app`)
- **框架**: Flutter 3.x + Dart 3.x
- **特点**: 原生移动应用体验，支持 iOS/Android
- **功能**: 拍照识别、完整错题管理、数据同步
- **状态管理**: Riverpod

## 🚀 快速开始

### Web 版
直接在浏览器中打开 `web/index.html` 即可使用。

在线演示：http://1.117.75.155/ai-news/cuoti-web.html

### Flutter 版
```bash
cd cuoti_app
flutter pub get
flutter run
```

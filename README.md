# QuizMaster - 智能错题本

一个基于 Web 的智能错题管理系统，帮助学生高效记录、分类和复习错题。

## 功能特性

- 📷 **拍照记录** - 支持图片上传，快速记录错题
- 📂 **分类归档** - 按科目（数学/英语/语文/物理/化学）智能分类
- 📝 **智能组卷** - 根据错题自动生成复习试卷
- 📊 **学习统计** - 实时追踪学习进度和掌握情况
- 💾 **本地存储** - 数据保存在浏览器本地，保护隐私

## 技术栈

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- LocalStorage API

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

## 界面预览

暗黑护眼主题，采用墨黑 + 霓虹青的配色方案。
